# DA Line Follower Robot Contest – Official Rules

## Overview

Build a robot that follows a black line on a white surface. Fastest time wins.  
This is an autonomous-only competition held at De Anza College.

---

## Robot Specifications

### Size
- Max footprint: **8 x 8 inches (~20.3cm x 20.3cm)** (no height limit)
- Measured at rest before the run
- Robots exceeding the footprint are disqualified before they run

### Power
- Max voltage: **12.6V** (3S LiPo maximum)
- Any lower voltage battery is allowed
- No power supply from external sources during a run

### Autonomy
- Robot must be fully autonomous — no remote control, no Bluetooth input, no human intervention once started
- Allowed: onboard microcontrollers (Arduino Nano, ESP32, STM32, etc.)

---

## Track Specifications

- Surface: white mat or paper
- Line: black tape or a printed black line, **3/4 to 1 inch wide** (~1.9–2.5 cm)
- Track includes: straight sections, curves, and at least one sharp turn
- Track layout is revealed on the day of competition

---

## Competition Format

### Runs
- Each robot gets **3 timed runs**
- Best time counts toward the final ranking
- If a robot fails to complete the track in all 3 runs, it is ranked by furthest point reached

### Timing
- Timer starts when the robot crosses the start line
- Timer stops when the robot crosses the finish line
- If the robot leaves the track and does not recover within **10 seconds**, the run ends and is marked DNF (Did Not Finish)

### Start Procedure
1. Place robot behind the start line
2. Operator may press a single start button
3. No further input allowed

---

## Scoring

| Place | Criteria |
|-------|----------|
| 1st (Sprinter) | Fastest single run time |
| 2nd, 3rd | Second and third fastest times |
| Innovation Award | Judged by organizers — best chassis design or sensor integration |

Ties are broken by the second-best run time.

---

## Penalties

| Violation | Penalty |
|-----------|---------|
| Robot leaves track, self-recovers | +5 sec per instance |
| Robot requires manual reset | Run marked DNF |
| Robot exceeds size limits | Disqualified from event |
| Remote/wireless control used | Disqualified from event |

---

## Check-In and Inspection

- Robots must pass a size and power inspection before competing
- Bring your robot at least **30 minutes before** your scheduled run
- Organizers measure footprint with a physical gauge — if it doesn't fit, you don't run

---

## Allowed Components

- Any microcontroller (Arduino, ESP32, Raspberry Pi, STM32, etc.)
- Any motor driver and motor type
- Any sensor type (IR, camera, TOF, etc.)
- 3D printed, laser cut, or off-the-shelf chassis

---

## Prohibited

- Remote control input of any kind during a run
- Batteries exceeding 12.6V
- Robots that damage the track surface
- Robots wider or longer than 8 inches (~20.3cm) at the footprint

---

## Innovation Award Criteria

Judges look for:
- Unusual or clever chassis design
- Creative sensor layout or sensor fusion
- Unique programming approach (e.g. machine learning, adaptive PID)
- Clean and well-documented build

You can bring a short demo or explain your design to the judges before your run.

---

## Code of Conduct

- Help each other out — this event is about learning
- No sabotage of other robots or equipment
- Organizers have final say on all rule disputes
